Hoshi — Home Services Platform

On-demand home services platform connecting users with trusted technicians — built for speed, trust, and seamless service delivery.

Flutter Firebase REST APIs Cubit Real-time Chat
My Role: Mobile Team Lead & App Manager @ Novix Code

Overview

Hoshi is a full-service home maintenance platform designed to simplify how users request and manage home services — from electrical repairs to cleaning and landscaping.

Target Users: Homeowners & busy individuals

Core Value: Instant service requests, competitive pricing, and seamless communication with technicians.

The Problem

Finding reliable home service providers is often slow, unstructured, and lacks transparency. Users struggle with unclear pricing, poor communication, and inconsistent service quality.

The Solution

We built Hoshi as a scalable mobile platform that allows users to submit service requests in seconds, receive multiple offers from technicians, and choose based on price and ratings.

As Mobile Team Lead, I managed architecture decisions, team workflows, and delivery quality while ensuring performance and scalability.

Key Features

Multi-service request system (electric, plumbing, cleaning)
Upload images & detailed descriptions
Real-time technician offers & pricing
In-app chat after request acceptance
Rating & review system
Simple and fast UX

Technical Implementation

  • Flutter for cross-platform mobile development
  • Cubit (Bloc) for scalable state management
  • REST APIs integration using Dio
  • Firebase for authentication & real-time features
  • Modular clean architecture for maintainability

Results & Impact

⚡ Fast Service Requests in Seconds
📈 Higher User Engagement
🤝 Better Service Transparency
